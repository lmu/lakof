lakof.theme Installation
------------------------

To install lakof.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``lakof.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        lakof.theme
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
